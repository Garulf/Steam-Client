# Steam Client

Steam Client is a Python library for interacting with a local Steam installation.

Use this site to quickly get started, see common usage patterns, and browse API docs generated from real source modules in this repository.

## What you can do

- Read installed library folders and games
- Access game metadata and artwork paths
- Read local users and non-Steam shortcuts
- Launch Steam URI commands
